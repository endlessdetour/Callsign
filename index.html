Before you start

Two-minute prep checklist

Have these ready before running the installer — each item links to a step-by-step guide.

1

A domain + a Linux server, pointed at each other

Rent any small VPS (1 vCPU / 1 GB is plenty) and register a domain, then add a DNS A record pointing your domain to the server's public IP.

How to add a DNS A record →
2

A Debian/Ubuntu server, root access, ports 80 & 443 open

The installer uses apt, so use Debian or Ubuntu, and run it with sudo. Open inbound TCP 80 and 443 in your cloud firewall / security group — port 80 is required for the automatic Let's Encrypt certificate.

How to open firewall ports (UFW) →
3

Recommended: put Cloudflare in front to hide your server IP

Move your domain's nameservers to Cloudflare and enable the orange-cloud proxy, then set SSL/TLS mode to Full (strict). Visitors only ever see Cloudflare's IPs — your origin stays hidden and DDoS-shielded. Tip: keep the record DNS-only for the first install so the certificate can be issued, then turn the proxy on.

Get started

Deploy in 3 steps

From zero to a running private network in three short steps.

1

Run the installer

One command on any Linux server provisions and starts everything.

2

Open the admin console

Visit https://<your-domain>/login, set your admin password, and create a user.

3

Connect the client

Download the client, paste the access token, and you're on the network.

wget -qO- https://raw.githubusercontent.com/endlessdetour/Callsign/main/deploy/install-server.sh | sudo bash
Hardened

Security you don't have to configure

Sensible, strong defaults — fully documented if you want the details.

✓

pbkdf2_sha256 (120k rounds) + per-user salt, constant-time compare, timing-equalized login.

✓

Server-stored session tokens with a per-device IP lease — no address collisions.

✓

Tunnel source-IP anti-spoofing and an optional Cloudflare-only origin gate.

✓

Security headers (CSP, HSTS, X-Frame-Options) on every response; server_tokens off.

✓

DPAPI-encrypted client access token at rest on Windows.

✓

Forced first-login credential change; seeded admin disabled afterward.
