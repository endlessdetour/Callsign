What you get

Everything you need, nothing you don't

A lightweight overlay network with the management tools built right in.

🛰️

Overlay tunnel

WebSocket control & tunnel planes with per-device session tokens and a Linux tun data path.

🎛️

Admin console

Role-based web UI to manage users, tokens, expiry, and access — with live system health at a glance.

🔐

Secure by default

Hashed credentials, anti-spoofing, DPAPI-encrypted client tokens, and hardened nginx out of the box.

⚡

One-command install

A single script provisions TLS, nginx, systemd units, and the seed admin — then starts everything.

Get started

Deploy in 3 steps

From zero to a running overlay network in three short steps.

1

Run the installer

One command on any Linux server provisions and starts everything.

2

Open the admin console

Visit https://<your-domain>/login, set your admin password, and create a user.

3

Connect the client

Download the client, paste the access token, and you're on the network.

wget -qO- https://raw.githubusercontent.com/endlessdetour/Callsign/fast_iteration/deploy/install-server.sh | sudo CALLSIGN_BRANCH=fast_iteration bash
Hardened

Security you don't have to configure

Sensible, strong defaults — fully documented if you want the details.

✓

pbkdf2_sha256 (120k rounds) + per-user salt, constant-time compare, timing-equalized login.

✓

Server-stored session tokens with per-device overlay IP leases — no address collisions.

✓

Tunnel source-IP anti-spoofing and an optional Cloudflare-only origin gate.

✓

Security headers (CSP, HSTS, X-Frame-Options) on every response; server_tokens off.

✓

DPAPI-encrypted client access token at rest on Windows.

✓

Forced first-login credential change; seeded admin disabled afterward.
